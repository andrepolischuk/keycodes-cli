# keycodes-cli

  > Key codes and names

  [keycodes][keycodes] - API for this module

## Install

```sh
npm install --global keycodes-cli
```

## Usage

```sh
keycodes --help

  Usage
    keycodes <key>

  Examples
    keycodes enter
    keycodes 13 --name

  Options
    -n, --name  get key name
```
